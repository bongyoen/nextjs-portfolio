# Beyondi - 포트폴리오

## 개요

**개발** 관련  **포트폴리오**

---

## 기술 스택

- `NextJs`, `TypeScript`, `Tailwind CSS`, `Oracl Cloud Plaform`, `Postgres`, `Docker Compose`, `Git Action`

---

## 주요 기능

- `Tailwind CSS` 기반 반응형 UI
- `Dark` 모드 지원
